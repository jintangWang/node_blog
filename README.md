# node_blog

> 跟着这位大神学习的`node`项目[链接](https://github.com/nswbmw/N-blog)

## 运行此项目

``` bash
# clone respository
git clone git@github.com:jintangWang/node_blog.git

# enter the directory
cd node_blog

# install dependencies
npm install

# serve with hot reload at localhost:8082
npm run start

```

**Tip:**需要在本地安装`mongodb`才可以正确启动
